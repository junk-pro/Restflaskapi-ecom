This is an e commerce api for a store in which users can be authenticated to add and delete products from 
the database. It is made by using python and flask framework.
The data base used in this project is SQLAlchemy.
It has endpoints to add/delete and update products.



To run the following program it is advisable to get all the pre requisite installed first in a virtual environment. 
Install all the required dependecies from the the command prompt by typing the following command
pip install -r requirements.txt
